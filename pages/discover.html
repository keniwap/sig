About CODEX

CODEX is a student-led Special Interest Group focused on software engineering, web development, and collaborative learning. We host weekly sessions, hands-on workshops, and project challenges to help members grow practical skills.

  • Hands-on coding workshops
  • Peer mentoring and study groups
  • Project showcases and code reviews
  • Industry talks when available
